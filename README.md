# PurseToken Subgraph
Subgraph for Purse Token on [pursetoken.com](www.pursetoken.com)

## Getting started
### 1. Clone repository
```
git clone https://github.com/FunctionX-SG/PurseToken_Subgraph.git
```

### 2. Install and run codegen
```
yarn install

yarn codegen
```

### 3. Build
```
yarn build
```
